Capítulo 3 Variables y Constantes

Variables:

  • Las variables son espacios de memoria que almacenan información y pueden modificarse a lo largo de la ejecución del programa. Se declaran usando la palabra clave let (o var para compatibilidad, aunque se recomienda let). Se destacan aspectos como la nomenclatura (camelCase) y la importancia de respetar mayúsculas y minúsculas.

    • Ejemplo:

    let nombreVariable = "valor";

Constantes:

  • Las constantes son similares a las variables pero, una vez asignado un valor, no pueden modificarse. Se declaran con la palabra clave const. Por convención, se suelen escribir en mayúsculas y con guiones bajos para los nombres compuestos.

    • Ejemplo:

    const NOMBRE_CONSTANTE = "valor fijo";
